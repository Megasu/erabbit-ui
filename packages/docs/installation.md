# Get Start

## Installation

Erabbit UI you can install it with:

::: code-group

```sh[npm]
npm i erabbit -S
```

```sh[pnpm]
pnpm i erabbit -S
```

```sh[yarn]
yarn add erabbit -S
```

```sh[bun]
bun add erabbit -S
```

:::

## What is Erabbit UI ?
